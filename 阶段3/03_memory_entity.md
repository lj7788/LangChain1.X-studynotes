# 03_memory_entity.py

## 功能说明

`EntityMemory` 是 LangChain 1.x 中的实体记忆组件，能够自动从对话中提取和记忆实体信息（如人物、地点、组织等）。

## 核心特性

- **自动提取实体**: 使用 LLM 自动识别对话中的实体
- **实体关联**: 记住实体之间的关系
- **持久化存储**: 实体信息可以保存和加载

## 关键参数

| 参数 | 说明 |
|------|------|
| llm | 用于提取实体的 LLM |
| memory_key | 在 prompt 中引用实体记忆的变量名 |
| return_messages | True 返回消息列表，False 返回字符串 |

## 使用场景

- 记住用户提到的所有人名、地名、机构名
- 记住实体之间的关系
- 在长对话中保持对关键信息的追踪

## 运行示例

```bash
python 阶段3/03_memory_entity.py
```

## 输出示例

```
=== 对话 1：提到人物 ===
用户: 我的朋友李明在北京工作，他是一名医生。
助手: 好的，我已经记住你的朋友李明的信息了。他在北京工作，是一名医生。

=== 查看实体记忆 ===
实体信息:
- 李明：在北京工作的医生，用户的朋友

=== 对话 2：提到更多人物 ===
用户: 李明的妻子叫王芳，她是一名老师。他们住在上海。
助手: 好的，我已经记住王芳的信息了。她是李明的妻子，是一名老师，住在上海。

=== 对话 3：询问实体信息 ===
用户: 告诉我你知道的关于李明的所有信息
助手: 根据我记住的信息：
- 李明是用户的朋友
- 在北京工作
- 是一名医生
- 他的妻子叫王芳

=== 对话 4：询问另一个实体 ===
用户: 王芳是做什么工作的？
助手: 王芳是一名老师，是李明的妻子。
```
